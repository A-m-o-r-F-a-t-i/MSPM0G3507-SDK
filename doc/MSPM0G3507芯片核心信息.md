# MSPM0G3507 芯片核心信息

## 1. 基本规格

| 参数 | 规格 |
|------|------|
| **内核** | Arm® 32位 Cortex®-M0+ CPU |
| **最高频率** | 80MHz |
| **闪存** | 128KB（带ECC纠错） |
| **SRAM** | 32KB（带ECC/奇偶校验） |
| **工作电压** | 1.62V 至 3.6V |
| **工作温度** | -40°C 至 125°C |
| **存储器保护** | MPU（8个可编程区域） |

---

## 2. 核心外设汇总

| 类别 | 外设 | 数量/规格 |
|------|------|-----------|
| **ADC** | 12位同步采样ADC | 2个，4Msps，最多17通道 |
| **DAC** | 12位数模转换器 | 1个，1Msps，带输出缓冲器 |
| **OPA** | 零漂移斩波运算放大器 | 2个，0.5µV/°C漂移，最高32x增益 |
| **COMP** | 高速比较器 | 3个，8位基准DAC，32ns传播延迟 |
| **GPAMP** | 通用放大器 | 1个 |
| **定时器** | 高级控制定时器(TIMAx) | 2个，16位 |
| | 通用定时器(TIMGx) | 5个（含QEI/HALL支持） |
| **通信** | UART | 4个（UART0扩展，UART1-3主要） |
| | I2C | 2个（最高1Mbit/s） |
| | SPI | 2个（最高32Mb/s） |
| | CAN-FD | 1个 |
| **其他** | DMA | 7通道 |
| | RTC | 带警报和日历模式 |
| | WWDT | 2个窗口看门狗 |

---

## 3. 时钟系统

| 时钟源 | 频率 | 说明 |
|--------|------|------|
| **SYSOSC** | 4-48MHz | 内部系统振荡器，可校准 |
| **SYSPLL** | 最高80MHz | 系统锁相环 |
| **LFOSC** | 32.768kHz | 内部低频振荡器 |
| **HFXT** | 4-48MHz | 外部高频晶体 |
| **LFXT** | 32kHz | 外部低频晶体 |

**时钟树：**
- **SYSCLK**：系统时钟，最高80MHz
- **HCLK**：总线时钟，最高80MHz
- **PCLK**：外设时钟，最高80MHz
---

## 4. 模拟外设关键参数

### ADC
- 分辨率：12位（硬件均值可达14位有效）
- 采样率：4Msps
- 基准：内部1.4V/2.5V或外部

### OPA
- 类型：零漂移斩波稳定
- 漂移：0.5µV/°C
- 增益：1x至32x可编程
- 压摆率：1.3-4.9 V/µs

### DAC
- 分辨率：12位
- 采样率：1Msps
- 压摆率：5.5 V/µs

## 6. 存储器映射关键区域

| 区域 | 地址范围 | 说明 |
|------|----------|------|
| **闪存** | 0x0000.0000 - 0x0001.FFF8 | 128KB代码区 |
| **SRAM** | 0x2000.0000 - 0x200F.FFFF | 32KB数据区 |
| **外设** | 0x4000.0000 - 0x40FF.FFFF | 外设寄存器 |
| **系统** | 0xE000.0000 - 0xE00F.FFFF | Cortex-M0+系统寄存器 |

---

## 7. 电源与功耗

| 模式 | 功耗特性 |
|------|----------|
| **运行模式** | 96µA/MHz (CoreMark) |
| **睡眠模式** | CPU暂停，外设可运行 |
| **停止模式** | PD1断电，PD0保持 |
| **待机模式** | 极低功耗，可唤醒 |
| **关断模式** | <1µA，仅可通过NRST/SWD/IO唤醒 |

**电源域架构：**
- **PD1**：CPU、存储器、高性能外设（可断电节能）
- **PD0**：低速低功耗外设（始终供电）
